## High-Impact Self-Introduction System

> Structured communication under strict time constraints (5-minute window)

---

## 📌 Overview

This project focused on designing a self-introduction as a structured system,  
rather than a simple presentation.

The goal was to maintain clarity and message retention  
within a limited time frame.

---

## 📌 Problem

- Information was scattered and lacked prioritization  
- No clear narrative flow across slides  
- Visual elements often distracted from core messages  
- Key points were not retained after delivery  

---

## ⚙️ Approach

- Treated the presentation as a **signal delivery system**  
- Focused on three core areas:
  - Message structure (clarity)
  - Attention control (focus)
  - Narrative flow (continuity)  
- Reduced decision overhead during presentation  
- Prioritized high-impact signals over full explanations  

---

## 🛠 Implementation

### 1. Structural Design
- Built a linear flow: Introduction → Core → Conclusion  
- Compressed content into key phrases  
- Removed redundant explanations  

### 2. Visual System
- Applied a dark + minimal theme  
- Reduced visual noise to improve readability  
- Used contrast and spacing to guide attention  

### 3. Message Optimization
- Converted sentences into action-oriented phrases  
- Maintained consistency across slides  
- Focused on signal clarity over completeness  

---

## 💡 Insights

- Communication is a structural problem, not a volume problem  
- Visual design functions as a control layer, not decoration  
- Message retention depends on signal clarity, not information quantity  
- Even simple presentations benefit from system-level thinking

## 🔍 Reflection

This project demonstrated that effective communication  
is not achieved through adding more information,

but through removing everything unnecessary.

It required:
- Intentional structure design  
- Controlled information flow  
- Consistent signal prioritization  

This led to a simple principle:

> Don’t present information — engineer how it is delivered.
